Top 8 Web Development Trends 2018

Ada 8 trend web development yang tidak diurutkan berdasarkan kepentingannya. Mereka adalah sebagai berikut:

The Basic: HTML, CSS, JavaScript

Ketiga hal ini adalah yang paling basic atau mendasar untuk Web Development. Untuk menjadi seorang web developer maka kamu harus mengerti dasarnya. JavaScript menguasai semuanya. Tapi belajar JavaScript saja tidak cukup, karena kamu harus mengerti HTML dan CSS. Ibartnya sebuah kepala, HTML adalah tengkorak, CSS adalah kulit, warna wajah, mata, hidung, dll. Sedangkan JavaScript adalah senyuman :) dan kedipan mata.

NodeJs

NodeJs bukanlah hal yang baru dan sudah trending sejak 2 tahun terakhir. NodeJs digunakan di front-end dengan bahasa paling populer, JavaScript. NodeJs juga bisa digunakan di back-end. Dan karena bahasa yang digunakan adalah JavaScript maka akan mudah dipelajari karena sebelumnya kamu sudah mendapatkan basic JavaScript, bukan? Node.js is simply JavaScript yang dapat digunakan di client-side maupun server-side. Jadi, Node.Js ini sangat penting untuk dipelajari.

Front-End FrameWork

Vue, Angular, and React

React adalah sebuah Library. Ketiga hal di atas menciptakan pengalaman pengguna/user experience yang sangat menarik untuk aplikasi web kamu karena semuanya dikontrol dengan JavaScript. Pengguna mungkin terus menyukai aplikasi web yang menarik, sehingga kerangka kerja frontend seperti Angular, React atau Vue.js akan tetap penting di tahun 2018. Mempelajari salah satu atau semuanya akan menjadi ide yang sangat bagus. Mereka memungkinkan dirimu untuk belajar tentang web developmnet dan menciptakan web yang bagus dengan interface yang interaktif.

Web Components/Compiler

Sampai akhir 2017, dukungan browser adalah masalah yang lebih besar namun pada awal 2018, ini terlihat lumayan. Empat API Web yang membentuk komponen web - Unsur Khusus, DOM Shadow, Template HTML dan Impor HTML - didukung oleh semua browser terkenal.

Gagasan di balik web components itu sederhana: Mengapa kita harus menggunakan perpustakaan seperti React untuk membangun komponen yang hanya berfungsi di perpustakaan itu dan tidak hanya membangun komponen ini dengan vanilla JavaScript dan menggunakannya juga di luar kerangka kerja?

Static Web Pages

Berikutnya yang trend di tahun 2018 adalah Web page statis. Static Web Pages diolah oleh Jekyll, Hugo, atau Nuxt.js. Semua alat dan proyek ini memungkinkan kita melakukan satu hal: Buatlah halaman web yang kemudian kita buat di server lokal kita atau beberapa server build. Kita kemudian menempatkan sekumpulan file statis - HTML, CSS dan JavaScript. Selesai!

Progressive Web Apps

Progresif Web Apps (PWA) adalah istilah yang merangkum beberapa teknologi yang memungkinkan kita untuk membangun aplikasi web yang berperilaku dan tersasa seperti aplikasi mobile.

Terdapat offline support, push notification, background sync ke dalam web page normal yang mana tidak terinstall di app store.

Serverless

Serverless bukan berarti tidak ada server, akan tetapi kode server-sidemu dapat dikirim ke server seperti AWS Lamda dimana kamu tidak perlu memikirkan skalabilitas karena kamu tidak mengatur server, kamu tidak perlu khawatir tentang konfigurasi server dengan benar. Kamu cuma kirim kodenya dan menjalankan itu ketika membutuhkannya.

Kamu juga hanya perlu membayar untuk yang kamu butuhkan, bukan yang kamu perkirakan. Serverless juga memungkinkan kamu untuk fokus hanya kepada codingan dan tidak perlu menjadi admin sistem. Karena serverless menyelesaikan masalah keamanan, biaya, dan perencanaan. Serta memeberikanmu banyak waktu untuk diluanggkan untuk codinganmu.

AI/Bots

Artificial Intellegence dan Bots menjadi lebih mudah diakses untuk semua orang. Kamu tidak perlu menjadi seorang Data Scientist. Untuk mulai menggunakan AI di dalam aplikasimu, sama seperti serverless, akan ada lebih banyak lagi service yang akan tersedia oleh AWS atau Google yang memudahkanmu untuk menggabungkan beberapa features yang dijalankan oleh AI ke dalam aplikasimu seperti image recognition.

Keputusan yang diambil BOTs dan AI dapat meningkatkan banyak model bisnis
